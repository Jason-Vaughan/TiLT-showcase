## What is TiLT?

TiLT is a full-stack web application that automates union-compliant time tracking and pay calculations for IATSE members. It replaces manual spreadsheets with a contract-aware system that understands CBA rules, overtime thresholds, meal penalties, and benefits eligibility — automatically.

The system started as a personal tool to track my own hours at IATSE Local 16 and evolved into a production platform with active users, a configurable rules engine, and an interactive tour.

## Features

- **Contract-Aware Calculations** — Configurable CBA rules engine supporting 12 rule types: daily OT/DT thresholds, meal penalties, short turn penalties, nth-day-worked premiums, holiday pay, and more
- **Multi-CBA Support** — Parent/child CBA relationships with a visual setup wizard
- **Real-Time Pay Tracking** — See calculated gross pay, track payments received vs. pending, view breakdowns by job
- **IATSE Web Tools Integration** — Auto-import job assignments from your union portal
- **Complete Audit Trails** — Every calculation logged with before/after snapshots
- **Benefits Eligibility Tracking** — Quarterly benefits hour tracking
- **Mobile-First Design** — Responsive across desktop, tablet, and phone
- **Interactive Tour** — Full-featured demo with real data so you can try before you commit
- **Privacy-First** — Full encryption, granular sharing controls, your data stays yours
